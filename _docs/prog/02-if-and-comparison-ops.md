## –¢–∏–ø –¥–∞–Ω–Ω—ã—Ö `bool`

–û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∑–Ω–∞—á–µ–Ω–∏—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–≥–æ –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ —Ç–∏–ø–∞ bool. –ó–Ω–∞—á–µ–Ω–∏—è –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ —Ç–∏–ø–∞ –º–æ–≥—É—Ç –ø—Ä–∏–Ω–∏–º–∞—Ç—å –æ–¥–Ω–æ –∏–∑ –¥–≤—É—Ö –∑–Ω–∞—á–µ–Ω–∏–π: `True (–∏—Å—Ç–∏–Ω–∞)` –∏–ª–∏ `False (–ª–æ–∂—å)`.

–ï—Å–ª–∏ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –ª–æ–≥–∏—á–µ—Å–∫–æ–µ `True` –∫ —Ç–∏–ø—É `int`, —Ç–æ –ø–æ–ª—É—á–∏—Ç—Å—è `1`, –∞ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ `False` –¥–∞—Å—Ç `0`. –ü—Ä–∏ –æ–±—Ä–∞—Ç–Ω–æ–º –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–∏ —á–∏—Å–ª–æ `0` –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç—Å—è –≤ `False`, –∞ –ª—é–±–æ–µ –Ω–µ–Ω—É–ª–µ–≤–æ–µ —á–∏—Å–ª–æ –≤ `True`. –ü—Ä–∏ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–∏ `str` –≤ `bool` –ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞–µ—Ç—Å—è –≤ `False`, –∞ –ª—é–±–∞—è –Ω–µ–ø—É—Å—Ç–∞—è —Å—Ç—Ä–æ–∫–∞ –≤ `True`.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–∏–º–µ—Ä–æ–≤:

```python
>>> else:
>>> 	print('–í—ã –Ω–µ –≤–≤–µ–ª–∏ —Å–≤–æ—ë –∏–º—è!')
```


> –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ `True` –∏–ª–∏ `False` –ø–∏—à—É—Ç—Å—è —Å
> **–±–æ–ª—å—à–æ–π –±—É–∫–≤—ã**. –ï—Å–ª–∏ –Ω–∞–ø–∏—Å–∞—Ç—å –∏—Ö —Å –º–∞–ª–µ–Ω—å–∫–æ–π, —Ç–æ python –ø–æ–¥—É–º–∞–µ—Ç, —á—Ç–æ —ç—Ç–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è, –ø–æ–ø—ã—Ç–∞–µ—Ç—Å—è –µ—ë –Ω–∞–π—Ç–∏ –∏ —Å–ª–æ–º–∞–µ—Ç—Å—è, –∫–æ–≥–¥–∞ –Ω–µ –Ω–∞–π–¥–µ—Ç

## –õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

–ï—Å–ª–∏ –º—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –¥–≤–∞ –∏–ª–∏ –±–æ–ª–µ–µ —É—Å–ª–æ–≤–∏–π –∑–∞ —Ä–∞–∑, –º—ã –º–æ–∂–µ–º –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞–º–∏ `and`, `or` –∏–ª–∏ `not`. –í–æ—Ç –∫–∞–∫ –æ–Ω–∏ —Ä–∞–±–æ—Ç–∞—é—Ç:

`and` (–ª–æ–≥–∏—á–µ—Å–∫–æ–µ –ò) –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏—Å—Ç–∏–Ω—É (`True`) —Ç–æ–ª—å–∫–æ –≤ —Å–ª—É—á–∞–µ –µ—Å–ª–∏ –æ–±–∞ —É—Å–ª–æ–≤–∏—è –ø–æ –æ—Ç–¥–µ–ª—å–Ω–æ—Å—Ç–∏ –≤–µ—Ä–Ω—ã (—Ç–æ–∂–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç `True`)<br>
`or` (–ª–æ–≥–∏—á–µ—Å–∫–æ–µ –ò–õ–ò) –≤–µ—Ä–Ω–µ—Ç –∏—Å—Ç–∏–Ω—É –≤ —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ –∏–∑ —É—Å–ª–æ–≤–∏–π –≤–µ—Ä–Ω–æ.<br>
`not` (–ª–æ–≥–∏—á–µ—Å–∫–æ–µ –ù–ï–¢) –≤–æ–∑—å–º–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç —É—Å–ª–æ–≤–∏—è –∏ "–æ–±—Ä–∞—Ç–∏—Ç" –µ–≥–æ. –¢–æ –µ—Å—Ç—å, –µ—Å–ª–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç —É—Å–ª–æ–≤–∏—è `True`, —Ç–æ `not` –ø—Ä–∏–º–µ–Ω–µ–Ω–Ω—ã–π –∫ —ç—Ç–æ–º—É —É—Å–ª–æ–≤–∏—é –≤–µ—Ä–Ω–µ—Ç `False` –∏ –Ω–∞–æ–±–æ—Ä–æ—Ç.<br>

–î–∞–≤–∞–π—Ç–µ –ø–æ—Å–º–æ—Ç—Ä–∏–º –∫–∞–∫ —ç—Ç–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –Ω–∞ –ø—Ä–∏–º–µ—Ä–µ. –ö–æ–¥ –Ω–∏–∂–µ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç, —á—Ç–æ —Ö–æ—Ç—è –±—ã –æ–¥–Ω–æ —á–∏—Å–ª–æ –∏–∑ –¥–≤—É—Ö –Ω–∞—Ü–µ–ª–æ –¥–µ–ª–∏—Ç—Å—è –Ω–∞ 10 (–∫–æ–Ω—á–∞–µ—Ç—Å—è –Ω–∞ 0) –∏ –µ—Å–ª–∏ —Ç–∞–∫, —Ç–æ –ø–µ—á–∞—Ç–∞–µ—Ç **YES**, –∞ –µ—Å–ª–∏ –Ω–µ—Ç, —Ç–æ –ø–µ—á–∞—Ç–∞–µ—Ç **NO**:

```python
>>> a = int(input())
>>> b = int(input())
>>>
>>> if a % 10 == 0 or b % 10 == 0:
...     print('YES')
... else:
...     print('NO')
...
```

–ü—É—Å—Ç—å —Ç–µ–ø–µ—Ä—å –º—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ —á–∏—Å–ª–∞ a –∏ b –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å –µ—â–µ –∏ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –±–æ–ª—å—à–µ –Ω—É–ª—è:

```python
>>> a = int(input())
>>> b = int(input())
>>>
>>> if (a % 10 == 0 and a > 0) or (b % 10 == 0 and b > 0):
...     print('YES')
... else:
...     print('NO')
...
```

–ö–∞–∫ –≤–∏–¥–∏—Ç–µ, –º—ã –º–æ–∂–µ–º –Ω–µ —Ç–æ–ª—å–∫–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `and` –∏ `or` –≤ –æ–¥–Ω–æ–º `if`, –Ω–æ –∏ –≥—Ä—É–ø–ø–∏—Ä–æ–≤–∞—Ç—å —É—Å–ª–æ–≤–∏—è —Å–∫–æ–±–∫–∞–º–∏ –¥–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã —è–≤–Ω–æ –æ–±–æ–∑–Ω–∞—á–∏—Ç—å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —É—Å–ª–æ–≤–∏–π.

–ü–æ—Å–º–æ—Ç—Ä–∏–º –ø—Ä–∏–º–µ—Ä —Å `not`. –ü—É—Å—Ç—å –º—ã —Ö–æ—Ç–∏–º –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ —á–∏—Å–ª–æ a - –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ, –∞ —á–∏—Å–ª–æ b - –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–µ.
–≠—Ç–æ –º–æ–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –≤–æ—Ç —Ç–∞–∫–∏–º —É—Å–ª–æ–≤–∏–µ–º:

```python
>>> if a > 0 and not (b < 0):
...     pass
...
```

> –û–ø–µ—Ä–∞—Ç–æ—Ä `pass` –æ—á–µ–Ω—å –ø–æ–ª–µ–∑–µ–Ω, –∫–æ–≥–¥–∞ –Ω—É–∂–Ω–æ –Ω–∏—á–µ–≥–æ –Ω–µ –¥–µ–ª–∞—Ç—å. –ï—Å–ª–∏ –µ–≥–æ
> –Ω–µ –ø–æ—Å—Ç–∞–≤–∏—Ç—å, —Ç–æ –±—É–¥–µ—Ç —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∞—è –æ—à–∏–±–∫–∞. –ê —Ç–∞–∫, –∫–æ–¥ —Å—á–∏—Ç–∞–µ—Ç—Å—è –ø—Ä–∞–≤–∏–ª—å–Ω—ã–º!

–ö—Å—Ç–∞—Ç–∏, `not (b < 0)` –º–æ–∂–Ω–æ –±—ã–ª–æ –±—ã –∏ –∑–∞–º–µ–Ω–∏—Ç—å –Ω–∞ `b >= 0` –∏ –∫–æ–¥ –±—ã —Ä–∞–±–æ—Ç–∞–ª —Ç–æ—á–Ω–æ —Ç–∞–∫ –∂–µ.

## –ö–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è elif

–ò–Ω–æ–≥–¥–∞ –ø–∏—Å–∞—Ç—å –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ `if-else` –¥–æ–ª–≥–æ –∏ —É—Ç–æ–º–∏—Ç–µ–ª—å–Ω–æ, –æ—Å–æ–±–µ–Ω–Ω–æ –µ—Å–ª–∏ –ø—Ä–∏—Ö–æ–¥–∏—Ç—Å—è –ø—Ä–æ–≤–µ—Ä—è—Ç—å –º–Ω–æ–≥–æ —É—Å–ª–æ–≤–∏–π —Ä–∞–∑–æ–º. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –Ω–∞ –ø–æ–º–æ—â—å –ø—Ä–∏–¥–µ—Ç `elif` (—Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ –æ—Ç else if). –ü–æ —Å—É—Ç–∏ `elif` –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å—É—â–µ—Å—Ç–≤–µ–Ω–Ω–æ —É–ø—Ä–æ—Å—Ç–∏—Ç—å –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é –Ω–∏–∂–µ:

```python
>>> if a > 0:
...     pass
... else:
...     if b > 0:
...         pass
...
```

–ò —Å–¥–µ–ª–∞—Ç—å –µ–µ –≤–æ—Ç —Ç–∞–∫–æ–π:

```python

...     pass
... elif b > 0:
...     pass
...
```

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, –º—ã –∏–∑–±–∞–≤–∏–ª–∏—Å—å –æ—Ç –æ–¥–Ω–æ–≥–æ —É—Ä–æ–≤–Ω—è –≤–ª–æ–∂–µ–Ω–Ω–æ—Å—Ç–∏. –¢–æ –µ—Å—Ç—å, —Å–∞–º –∫–æ–¥ —Å—Ç–∞–ª –±–æ–ª–µ–µ —á–∏—Ç–∞–µ–º—ã–º, –Ω–æ –ø—Ä–∏ —ç—Ç–æ–º –Ω–∏—Å–∫–æ–ª—å–∫–æ –Ω–µ –ø—Ä–æ–∏–≥—Ä–∞–ª –≤ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏. –†–∞–∑—É–º–µ–µ—Ç—Å—è, –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ —Ç–∏–ø–∞ `if-elif` –º–æ–≥—É—Ç –∑–∞–≤–µ—Ä—à–∏—Ç—å—Å—è –∏ –±–ª–æ–∫–æ–º `else`, –Ω–∞–ø—Ä–∏–º–µ—Ä —Ç–∞–∫:

```python
>>> if a > 0:
...     pass
... elif b > 0:
...     pass
... elif c > 0:
...     pass
... else:
...     pass
...
```

## –ó–∞–¥–∞—á–∞: –∑–Ω–∞–∫ —á–∏—Å–ª–∞

–í –º–∞—Ç–µ–º–∞—Ç–∏–∫–µ –µ—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è [sgn](https://ru.wikipedia.org/wiki/Sgn), –ø–æ–∫–∞–∑—ã–≤–∞—é—â–∞—è –∑–Ω–∞–∫ —á–∏—Å–ª–∞. –û–Ω–∞ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —Ç–∞–∫: –µ—Å–ª–∏ —á–∏—Å–ª–æ –±–æ–ª—å—à–µ 0, —Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç 1. –ï—Å–ª–∏ —á–∏—Å–ª–æ –º–µ–Ω—å—à–µ –Ω—É–ª—è, —Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç -1. –ï—Å–ª–∏ —á–∏—Å–ª–æ —Ä–∞–≤–Ω–æ 0, —Ç–æ —Ñ—É–Ω–∫—Ü–∏—è –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç 0. –†–µ–∞–ª–∏–∑—É–π—Ç–µ –¥–∞–Ω–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é ‚Äî –¥–ª—è –≤–≤–µ–¥–µ–Ω–Ω–æ–≥–æ —á–∏—Å–ª–∞ –≤—ã–≤–µ–¥–∏—Ç–µ —á–∏—Å–ª–æ, –æ–ø—Ä–µ–¥–µ–ª—è—é—â–µ–µ –µ–≥–æ –∑–Ω–∞–∫. –ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å—Ä–∞–≤–Ω–µ–Ω–∏—è –∏ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é `if-elif-else`.

–í–æ–∑–º–æ–∂–Ω–æ–µ —Ä–µ—à–µ–Ω–∏–µ:

